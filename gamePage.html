4-In-A-Row

(About 250 lines of cancer, but it works. Use a browser with ES6.)

Updated 4/November

By: Dan Lynch

This prowser does not support EcmaScript 6's arrow function standard.
Consider using a good browser.

Take turns playing the game with someone else, either clicking on the slot you want to put the game token into, or by pressing the corresponding number on the keyboard. The first player to get four game tokens in a row, horizontally, vertically, or diagonally, wins!
